TEKNIK DESIGN WEB

Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Adaptive web adalah bentuk tipe website yang statis yang tidak berubah-rubah sekalipun dibuka melalui resulusi layar yang berbeda sedangkan responsive web design adalah kebalikannya dimana website responsive akan mengikuti sesuai ukuran layar dari user yang membuka website tersebut

Teknik ini sangat berguna terutama responsive layout karena akan memudahkan bagi para programmer serta lebih user friendly dari sisi user

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Benar, Hadirnya smartphone dan tablet telah mengubah dunia web desain secara signifikan karena pengguna sekarang rata-rata meggunakan smartphone untuk mengakses apapun bukan melalui komputer dekstop lagi

Teknologi akan berkembang tanpa bisa dibendung, begitu pula dengan web design dimasa depan, dimana ketika teknologi hologram, VR dan AR akan berkembang masif hal ini juga akan mempengarui web desain karena kemungkinan website akan bisa ditampilkan dalam bentuk tersebut

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Jika kekurangan fisiknya bukan dari panca indra penglihatan maka pengguna tersebut masih bisa melihat bentuk visual website tersebut sehingga informasi masih bisa disampaikan, namun jika panca indra penglihatannya yang tidak berfungsi hal ini akan menjadi kendala dan solusi untuk membuat website yang mendengarkan suara tentang isi dari website tersebut sangat dibutuhkan
