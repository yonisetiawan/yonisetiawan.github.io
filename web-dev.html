PERENCANAAN WEBSITE

Website apa yang paling ingin kamu buat? Tentang apa?

Saya ingin membuat sebuah website berbentuk B2C tentang Online Printing Service

Hasil atau informasi apa yang paling ingin kamu sampaikan?

Website tersebut akan menjadi media bagi percetakan-percetakan untuk menampilkan jasa cetak apa saja yang bisa dikerjakan oleh percetakan-percetakan tersebut, sehingga konsumen tinggal memilih jenis barang apa yang akan dicetak dari ribuan pengiklan percetakan tersebut dengan harga dan kwalitas yang bersaing

Bagaimana kira-kira kamu memproses pembuatan website dari awal hingga akhir?

Proses pembuatan Website saya bagi menjadi beberapa bagian

  1. Persiapan

    2. Mengumpulkan informasi tentang konsep B2C yaitu konsep dari website yang akan saya buat, serta menidentifikasi target pengguna sehingga bisa memberi gambaran seperti apa tampilan website akan dibuat

  2. Planning

    3. Membuat sitemap dan struktur website serta memilih bahasa pemrogramman yang akan digunakan

  3. Font End

    4. Membuat Visual Wireframe model website dengan memperhatikan UI dan UX serta mengatur konten-konten yang akan ditampilkan dala website dan kemudiann mengkonversi dalam bentuk kode HTML dan CSS

  4. Back End

    5. Seteleh front end website selesai dibuat saya akan membuat design database dan mulai memprogram dari sisi back end nya.

  5. Pengujian

    6. Melakukan pengujian website baik ke user terbatas maupun pengujian untuk membuka diberbagai browser serta platform yang berbeda kemudian memperbaiki bug yang terjadi dan melakukan pengujian kembali
